# Sistema de Análisis del Mercado de Automoción 🚗📊

Este proyecto presenta un análisis integral del mercado automovilístico utilizando técnicas avanzadas de **Big Data** e **Inteligencia Artificial**. El objetivo principal es extraer valor de grandes volúmenes de datos para predecir precios, analizar el sentimiento de los consumidores y detectar anomalías en el mercado.

## 📋 Descripción del Proyecto

El sistema procesa y analiza datos provenientes de múltiples fuentes para ofrecer una visión estratégica del sector. Se abordan desafíos como la predicción del valor residual de los vehículos, la clasificación de segmentos de mercado y la identificación de patrones de venta estacionales.

## 🚀 Funcionalidades y Minería de Datos

El proyecto implementa diversos modelos de aprendizaje automático:

* **Predicción de Valor Residual**: Uso de **Árboles de Decisión** para clasificar el valor futuro (Alto, Medio, Bajo) con una precisión del 91.6%.
* **Estimación de Precios**: Modelos de **Regresión Múltiple** y **Random Forest** para calcular el precio de venta óptimo.
* **Análisis de Sentimiento**: Clasificación de reseñas de clientes mediante **Naive Bayes** para entender la percepción del consumidor.
* **Detección de Anomalías**: Aplicación de algoritmos **K-NN** y **SVM** para identificar irregularidades o fraudes en los precios de mercado.
* **Segmentación**: Clasificación inteligente del segmento de mercado de los vehículos.

## 🛠️ Tecnologías y Metodología

- **Lenguajes y Herramientas**: Python (entorno de desarrollo para modelos de ML).
- **Procesamiento**: Técnicas de limpieza, integración de fuentes y tratamiento de valores anómalos.
- **Algoritmos**: 
  - Árboles de Decisión.
  - Random Forest.
  - Naive Bayes.
  - K-Nearest Neighbors (K-NN).
  - Support Vector Machines (SVM).

## 📈 Conclusiones del Análisis

- **Factores Clave**: La antigüedad del vehículo representa el **54.3%** de la importancia en la determinación del valor residual, seguida por la potencia y el mantenimiento.
- **Patrones de Venta**: Se identifica una estacionalidad clara, con picos de ventas en mayo tras la recuperación económica post-navideña.
- **Influencia Subjetiva**: El análisis demuestra que el valor de mercado depende mayoritariamente de características objetivas técnicas más que de opiniones subjetivas.
